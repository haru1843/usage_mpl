テキストを挿入する

Axes.text(x, y, s[, fontdict=None, withdash=False, **kwargs])のような書式です.

(自分が思う)特に重要なパラメータを以下に書いておきます.

引数名 内容 設定できる値
x x軸方向の位置
y y軸方向の位置
s 文字列
fontsize 文字の大きさ
color 文字色 いつもの
alpha 透明度 0.0~1.0
zorder 表示順序
verticalalignment, va どこを位置の起点とするか top, center, bottom, baseline, center_baseline
horizontalalignment, ha どこを位置の起点とするか left, center, right
bbox テキストを囲う図形の辞書
transform 位置の指定方法の設定 ax.transData, ax.transAxes, fig.transFigure 
import numpy as np
import matplotlib.pyplot as plt
import matplotlib as mpl

def main():
    font = {"family": "IPAexGothic"}
    mpl.rc('font', **font)
    
    fig = plt.figure()
    ax = fig.add_subplot(111)

    x = np.arange(0.01, 5, 0.01)
    ax.plot(x, np.log(x))

    # 文字列の描画
    ax.text(2, -1, "aaa", fontsize=20, alpha=0.4, color="C3")

    # haについて
    ax.vlines(4, ymin=-0.5, ymax=0.5, color="gray", linestyle="--", zorder=-10)
    ax.text(4, 0, "ha=right", ha="right")
    ax.text(4, 0, "ha=left", ha="left", color="red")
    ax.text(4, -0.2, "ha=center", ha="center", color="blue")

    # bboxについて
    ax.text(0.1, -2, "bbox", fontsize=15, color="red", bbox=dict(facecolor="gray", alpha=0.2))
    ax.text(1, 0, "テキストを\n目立たせる", fontsize=15, color="red", bbox=dict(facecolor="white", alpha=0.7, edgecolor="white"))

    # 表示位置の指定方法を変える
    ax.text(0.8, 0.2, "Axesで指定", color="blue", transform=ax.transAxes)
    ax.text(4.15, -4, "Dataで指定", color="blue", transform=ax.transData)
    

    plt.show()

if __name__ == '__main__':
    main()

haとva

haでは水平方向, vaでは垂直方向の基準点を設定することができます.

  • ha=leftなら, xで指定した点を文字列の左端に
  • ha=centerなら, xで指定した点を文字列の中央に
  • ha=rightなら, xで指定した点を文字列の右端に

という感じ.

vaも同様.

bbox

テキスト周辺を囲う図形の設定が行なえます.
bboxは辞書でスタイルを設定していきます.

bboxに設定できる値にどのようなものがあるか全然網羅していないので, 詳しくはないです.
(自分はもっぱらグラフと文字列がかぶらないようにする程度でしか使っていないので…)

transform

デフォルトだとx, yで指定する位置は, 図中のデータ範囲と同じになっています.

しかしtransformではその指定位置を違う方法で選ぶことが可能になります.

引数 内容
transform=ax.transAxes そのaxの縦横の位置を0.0~1.0としてセッティングできる.
transform=ax.transData デフォルトのやつ. データ範囲に合わせて設定する.
transform=fig.transFigure そのfigの縦横の位置を0.0~1.0としてセッティングできる.
