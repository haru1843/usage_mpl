arange

np.arange(start, end, step)でstart≤start+n∗step<end\rm start \leq start + {\it n}*step < endstart≤start+n∗step<endを満たすような数値を配列化してくれます. ほぼビルトイン関数のrangeと同じ動作ですが, 違う点としては以下の2点で,

  • 作成されるのはリストではなく, ndarrayという, numpyの配列である.
  • 各引数には小数点を利用できる

という部分である.

>>> a = np.arange(0, 10, 0.1)
>>> a.__class__.__name__
'ndarray'

numpy配列の特徴

プログラムを見て気づいたかもしれませんが, numpy配列に対して四則演算子を用いると, 要素毎にその演算がなされます.
また, 2つのnumpy配列間に四則演算子を用いると, それぞれの同じインデックスの要素ごとに演算がなされます.

import numpy as np

nda = np.arange(0, 3, 0.2)
print(nda)
print(3 * nda)

print()

nda1 = np.arange(0, 3, 0.3); print("       nda1 :", *map(lambda x: "{:4.3}".format(x), nda1))
nda2 = np.arange(1, 4, 0.3); print("       nda2 :", *map(lambda x: "{:4.3}".format(x), nda2))
print("nda1 + nda2 :", *map(lambda x: "{:4.3}".format(x), nda1+nda2))
print("nda1 * nda2 :", *map(lambda x: "{:4.3}".format(x), nda1*nda2))

関数 大まかな内容
numpy.random.rand(d0, d1, ..., dn) [0.0,1.0)[0.0, 1.0)[0.0,1.0)の範囲のランダムな小数を返す.
引数で戻ってくる配列の次元数を決められる.

他にも指数分布とか対数分布とかレイリー分布とかあるけど, そんなに使ったことないので, 使うときは調べてください.

boolで取り出す要素を指定する

numpy配列では, 通常のリストと同じように[]で, インデックスを指定したり, スライスを作成することができます.
しかし, それ以外にも, boot値でどの要素をとるかを選択できます.

対象の配列Aと同じ大きさの配列を用意し, 取りたい位置の要素をTrue, そうでない場合はFalseをいれ, 配列bool_indexを作成します.
その後, A[bool_index]のようにすると, Trueにした位置の要素のみ, 取り出すことができます.

import numpy as np

dna = np.arange(8)
print(dna)

bool_list = [False] * len(dna)
print(bool_list)

bool_list[2] = True
bool_list[5] = True
print(bool_list)

print(dna[bool_list])

配列Aに対し比較演算を行った結果resultをAのインデックスに指定すればよく, A[result]となります.
