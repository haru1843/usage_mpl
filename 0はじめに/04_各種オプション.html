グリッド線

Axesのヘルパーメソッドgrid()でグリッド線を追加できます.
そこまで引数として利用するものはないと思うので, とりあえず以下の2つだけ説明しておきます.

グリッドの色と透明度

いつもどおり, color(c)とalphaで設定できます.

import numpy as np
import matplotlib.pyplot as plt

def func(x):
  return np.sinc(x)

def main():
  fig = plt.figure()
  ax = fig.add_subplot(111)

  xlim = [-10, 10]

  x = np.linspace(*xlim, 1000)
  y = func(x)

  ax.plot(x, y)  # plot で 折れ線を描画

  ax.grid(c="red", alpha=1.0)
  ax.set_xlim(xlim)
  plt.show()

if __name__ == '__main__':  
  main()

方向の設定

axisで描画する方向を選択できます.

'both', 'x', 'y'のそれぞれの値を設定することができます.

線のスタイル, 太さ

いつもどおり, linestyle, lsで線のスタイル,
linewidth, lwで線の太さを変更できます.

こちらはplotのときと同じなので, 詳細は省略します.

import numpy as np
import matplotlib.pyplot as plt

def func(x):
  return np.sinc(x)

def main():
  fig = plt.figure()
  ax = fig.add_subplot(111)

  xlim = [-10, 10]

  x = np.linspace(*xlim, 1000)
  y = func(x)

  ax.plot(x, y, zorder=100, c="black", lw=10)  # plot で 折れ線を描画

  ax.grid(c="red", alpha=1.0, axis='x', lw=20)
  ax.grid(c="c", alpha=0.8, axis='y', ls='-.', lw=5)
  ax.set_xlim(xlim)
  plt.show()

if __name__ == '__main__':  
  main()

凡例

Axesのlegendメソッドで, labelのついた要素の凡例を表示することができます.

凡例の要素は基本的にいい感じのところにおいてくれます.
(凡例の位置を細かく変更するのは少しめんどいので, ここでは省略します.)

import numpy as np
import matplotlib.pyplot as plt

def main():
  fig = plt.figure()
  ax = fig.add_subplot(111)

  xlim = [-10, 10]

  x = np.linspace(*xlim, 1000)

  ax.plot(x, np.sinc(x), label="$\\rm sinc(x)$")
  ax.plot(x, np.sin(x), label="$\\rm sin(x)$")
  ax.plot(x, np.cos(x), label="$\\rm cos(x)$")

  ax.grid()
  ax.legend()
  ax.set_xlim(xlim)
  plt.show()

if __name__ == '__main__':  
  main()

軸の設定

軸の範囲の設定

軸の範囲を設定するにはAxesのset_xlim, set_ylimを使っていきます.
それぞれx軸の軸範囲, y軸の軸範囲を設定することができます.
指定の方法は以下の2通りがあります.

  • set_xlim([min, max]), set_ylim([min, max])
  • set_xlim(bottom=min, top=max), set_xlim(bottom=min, top=max)

個人的には前者をよく使いますが, 上限か下限のどちらかを設定したいときは後者の方でないといけないので, そちらを利用します.

import numpy as np
import matplotlib.pyplot as plt

def func(x):
  return np.sinc(x)

def main():
  fig = plt.figure()
  ax = fig.add_subplot(111)

  x = np.linspace(-10, 10, 1000)
  y = func(x)

  ax.plot(x, y)  # plot で 折れ線を描画

  ax.grid()
  ax.set_xlim([-2, 6])
  ax.set_ylim(bottom=0)
  plt.show()

if __name__ == '__main__':  
  main()

軸目盛りの設定

軸目盛りを置く位置を設定

Axesのset_xticks, set_yticksでX軸/Y軸の軸目盛りを置く位置を設定できます.
引数はシーケンス型なら大丈夫です.

import numpy as np
import matplotlib.pyplot as plt

def func(x):
  return np.sinc(x)

def main():
  fig = plt.figure()
  ax = fig.add_subplot(111)

  xlim = [-10, 10]

  x = np.linspace(*xlim, 1000)
  y = func(x)

  ax.plot(x, y)

  ax.set_xticks(range(-5, 6))
  ax.set_yticks(np.arange(-0.3, 1.1, 0.1))

  ax.grid()
  ax.set_xlim(xlim)
  plt.show()

if __name__ == '__main__':  
  main()

各軸目盛りに表示する文字列の設定

Axesのset_xticklabels, set_yticklabelsで設定できます.
軸に設定した位置に順に文字列を割り当てていく感じです.

import numpy as np
import matplotlib.pyplot as plt
import matplotlib as mpl

def func(x):
  return np.sinc(x)

def main():
  font = {"family": "IPAexGothic"}
  mpl.rc('font', **font)

  fig = plt.figure()
  ax = fig.add_subplot(111)

  xlim = [-10, 10]

  x = np.linspace(*xlim, 1000)
  y = func(x)

  ax.plot(x, y)

  ax.set_xticks([-6, -3, 0, 3, 6])
  ax.set_xticklabels(["軸目盛り\n1つ目", "2つ目", "3つ目", "4つ目"])
  ax.set_yticks([-0.2, 0, 0.6, 1])
  ax.set_yticklabels(["軸目盛り\n1つ目", "2つ目", "3つ目", "4つ目"])

  ax.grid()
  ax.set_xlim(xlim)
  plt.show()

if __name__ == '__main__':  
  main()
