axとplt

axを使った書き方 (オブジェクト指向インターフェース)

axを利用すると, 例えばsin⁡\sinsin波を描画する時, 以下のように書くことができます.

from matplotlib import pyplot as plt
import numpy as np

x = np.arange(0, 10, 0.1)
y = np.sin(x)

fig, ax = plt.subplots()
ax.plot(x, y)

もしくは

from matplotlib import pyplot as plt
import numpy as np

x = np.arange(0, 10, 0.1)
y = np.sin(x)

fig = plt.figure()
ax = fig.add_subplot(111)
ax.plot(x, y)

のようにもかけます.

pltを使った書き方 (pyplotインターフェース)

plt(pyplot)を利用すると, このように書けます.

from matplotlib import pyplot as plt
import numpy as np

x = np.arange(0, 10, 0.1)
y = np.sin(x)

plt.figure(1)
plt.subplot(111)

plt.plot(x, y)

それぞれの違い

pltではあまり操作の対象を意識せず, スラスラと書くことができます.
それに対し, axではどれを選択しているかを明示的に宣言する必要があります.

pltは一瞬で書く場合には, 書く量も減って楽なんですが, 個人的には細かい部分まで設定できるaxの方が利便性が高いので, そちらの利用をおすすめします.

Artist

ここまで説明してきたグラフにおける各要素はすべてArtistに属しています.
このArtistはcontainersとprimitivesのもので構成されています.

イメージ的にはcontainersは容器で, primitivesはそこに入れられる要素のような感じです.

Artist
  │
  ├─containers : Figure
  │                └─Axes
  │                     └─Axis
  │                          ├─XAxis
  │                          ├─YAxis
  │                          └─Tick
  │
  └─primitives : Line2D, AxesImage, Legend, Annotation…

各containerが持つ属性とヘルパーメソッド

containerに対応するオブジェクトを追加/処理をしてくれるメソッドがあり, これらはヘルパーメソッドと言われています.

Axes

属性

Axesの属性は以下のようなものがあります.

Axes属性 概要
ax.artists Artistオブジェクトのリスト
ax.patch Axesの背景となるRectangleオブジェクトの1つ
ax.collections Collectionオブジェクトのリスト
ax.images AxesImageオブジェクトのリスト
ax.legends Legendオブジェクトのリスト(なくなったぽい)
ax.lines Line2Dオブジェクトのリスト
ax.patchess Patchオブジェクトのリスト
ax.texts Textオブジェクトのリスト
ax.xaxis XAxisオブジェクトの1つ
ax.yaxis YAxisオブジェクトの1つ

ヘルパーメソッド

axのヘルパーメソッドは以下のとおりです.

ヘルパーメソッド 生成されるArtist 格納場所
ax.annotate Annotate ax.texts
ax.bar Rectangle ax.patches
ax.errorbar Line2D & LineCollection ax.lines & ax.collections
ax.fill Polygon ax.patches
ax.hist Rectangle ax.patches
ax.imshow AxesImage ax.images
ax.legend Legend ax.legends
ax.plot Line2D ax.lines
ax.scatter PathCollection ax.collections
ax.text Text ax.texts

Axis属性 概要

Axis.minorTicks minor tickのラベルや位置, 線を管理するTickオブジェクトのリスト
